Frequently Asked Questions About CSS

Browse through the most frequently asked questions.

CSS stands for Cascading Style Sheets. It's a language that tells your web browser how to make your website look good. While HTML creates the basic structure of your page, CSS adds colors, fonts, spacing, and layout. Without CSS, all websites would look plain and boring - just black text on a white background.

There are three ways to add CSS to your HTML:

  • Internal CSS: Put your CSS code inside <style> tags in the <head> section of your HTML file.
  • External CSS: Create a separate .css file and link it to your HTML using <link rel="stylesheet" href="styles.css"> in the <head> section.
  • Inline CSS: Add CSS directly to HTML elements using the style attribute, like <p style="color: red;">.

The best practice is to use external CSS files because they keep your code organized and can be used across multiple pages.

A class can be used on multiple elements and starts with a dot (.) in CSS. For example: .my-class { color: blue; }

An ID should only be used once per page and starts with a hash (#) in CSS. For example: #my-id { color: red; }

Think of classes like a group of students wearing the same uniform, while an ID is like one person's unique student number.

There are several ways to center elements:

  • For text: Use text-align: center;
  • For block elements (like divs): Use margin: 0 auto; and make sure the element has a specific width.
  • Using Flexbox: Put display: flex; justify-content: center; align-items: center; on the parent container.
  • Using Grid: Put display: grid; place-items: center; on the parent container.

Flexbox and Grid are modern methods that work well for most centering needs.

Here are the most common reasons CSS doesn't work:

  • Spelling mistakes: Check that you spelled property names and values correctly
  • Missing semicolons: Each CSS rule should end with a semicolon (;)
  • Wrong file path: Make sure your HTML file can find your CSS file
  • Specificity issues: Another CSS rule might be overriding yours
  • Cached files: Try refreshing your browser or clearing the cache

Use your browser's developer tools (press F12) to check for errors and see which CSS rules are being applied.

The box model describes how CSS calculates the total size of an element. Every element is like a box with four parts:

  1. Content: The actual text or image inside
  2. Padding: Space between the content and the border
  3. Border: A line around the padding (can be invisible)
  4. Margin: Space outside the border, separating it from other elements

The total width = content width + padding + border + margin. Understanding this helps you control spacing and layout better.
